마우스 우클릭 : 화면의 회전
		내가 현재 보는 위치를 기준	( = 3인칭 시점)

마우스 좌클릭 + alt : 화면의 회전
		      화면의 중점을 기준

alt + ctrl + 마우스 좌클릭
마우스 가운데 클릭		 : 화면의 위치 이동

마우스 휠드래그 : 확대 축소

객체 더블클릭 -> 계층뷰에서
좌표 해당객체의 중점을 기준으로 좌표가 제일 잘보이는 모습으로 확대(축소)가 일어남

q, w, e, r, t : 뷰이동, 이동, 회전, 확대축소 단축키
